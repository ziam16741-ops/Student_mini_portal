## 📘 Student Web Portal (Mini Project)

A multi-page **Student Web Portal** developed as part of the Web Engineering Lab (CYS-463L). This project demonstrates the integration of **HTML, CSS, Bootstrap, and JavaScript** to build a fully functional and interactive web application.

---

## 🚀 Features

### 🔐 Registration Page

* Student registration form with:

  * Name, Email, Password, Confirm Password, Age, Gender
* Form validation using JavaScript:

  * Required field validation
  * Email format checking
  * Password matching
  * Numeric age validation
* Redirects to Dashboard after successful submission

---

### 🏠 Dashboard Page

* Personalized welcome message using JavaScript
* Displays student name (simulated/passed data)
* Bootstrap-based UI (Navbar & Cards)
* Button to show current Date & Time
* Navigation to:

  * Courses
  * Feedback
  * Home

---

### 📚 Course Selection Page

* List of courses using checkboxes:

  * Web Development
  * Artificial Intelligence
  * Database Systems
  * Mobile App Development
* Displays selected courses using alert

---

### 💬 Feedback Page

* Feedback form with:

  * Name
  * Comments
  * Rating (Dropdown)
* Validation for required fields
* Success message on submission

---

### 🌐 Home Page

* Main landing page of the portal
* Navigation bar with links to all pages
* Styled using CSS and Bootstrap
* Includes basic UI enhancements and hover effects

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (DOM Manipulation & Validation)

---

## 📌 Project Objective

This project aims to provide hands-on experience in:

* Creating multi-page web applications
* Implementing client-side validation
* Designing responsive layouts using Bootstrap
* Handling navigation and user interaction

---

## 💡 Note

This is a **mini academic project** developed for learning purposes and demonstrates core front-end development concepts.
